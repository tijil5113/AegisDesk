1. **Multiple Windows**: You can open multiple instances of apps - just click the app icon again
2. **Search Bar**: Type URLs directly to open them in the browser, or type app names to open apps
3. **Window Positions**: Window positions and sizes are remembered between sessions

5. **Data Persistence**: Everything is saved automatically using browser local storage

## Troubleshooting

**Apps not opening?**
- Check the browser console for errors (F12)
- Make sure all files are in the correct folders
- Try refreshing the page

**Data not saving?**
- Check if localStorage is enabled in your browser
- Some browsers block localStorage in private/incognito mode

**Weather not showing?**
- The weather app uses sample data (no API key required)
- Data loads with a slight delay for better UX

**AI not responding?**
- The AI works with local responses by default
- For enhanced capabilities, add an OpenAI API key in Settings
- Check your internet connection if using API
